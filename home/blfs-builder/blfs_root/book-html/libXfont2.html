Introduction to libXfont2

Package Information

  • Download (HTTP): https://www.x.org/pub/individual/lib/libXfont2-2.0.5.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/lib/libXfont2-2.0.5.tar.bz2

  • Download MD5 sum: bdf528f1d337603c7431043824408668

Installation of libXfont2

Install libXfont2 by running the following commands: 

packagedir=libXfont2-2.0.5
  docdir="--docdir=$XORG_PREFIX/share/doc/$packagedir"
  case $packagedir in
    libICE* )
      ./configure $XORG_CONFIG $docdir ICE_LIBS=-lpthread
    ;;

    libXfont2-[0-9]* )
      ./configure $XORG_CONFIG $docdir --disable-devel-docs
    ;;

    libXt-[0-9]* )
      ./configure $XORG_CONFIG $docdir \
                  --with-appdefaultdir=/etc/X11/app-defaults
    ;;

    * )
      ./configure $XORG_CONFIG $docdir
    ;;
  esac
  make
  #make check 2>&1 | tee ../$packagedir-make_check.log

Now as the root user: 

 make install
