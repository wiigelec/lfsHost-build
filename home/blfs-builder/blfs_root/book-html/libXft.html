libXft

Introduction to libXft

Package Information

  • Download (HTTP): https://www.x.org/pub/individual/lib/libXft-2.3.4.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/lib/libXft-2.3.4.tar.bz2

  • Download MD5 sum: 5004d8e21cdddfe53266b7293c1dfb1b

Installation of libXft

Install libXft by running the following commands: 

packagedir=libXft-2.3.4
  docdir="--docdir=$XORG_PREFIX/share/doc/$packagedir"
  case $packagedir in
    libICE* )
      ./configure $XORG_CONFIG $docdir ICE_LIBS=-lpthread
    ;;

    libXfont2-[0-9]* )
      ./configure $XORG_CONFIG $docdir --disable-devel-docs
    ;;

    libXt-[0-9]* )
      ./configure $XORG_CONFIG $docdir \
                  --with-appdefaultdir=/etc/X11/app-defaults
    ;;

    * )
      ./configure $XORG_CONFIG $docdir
    ;;
  esac
  make
  #make check 2>&1 | tee ../$packagedir-make_check.log

Now as the root user: 

 make install
