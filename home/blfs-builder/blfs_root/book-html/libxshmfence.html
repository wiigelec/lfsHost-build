libxshmfence

Introduction to libxshmfence

Package Information

  • Download (HTTP): https://www.x.org/pub/individual/lib/libxshmfence-1.3.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/lib/libxshmfence-1.3.tar.bz2

  • Download MD5 sum: 42dda8016943dc12aff2c03a036e0937

Installation of libxshmfence

Install libxshmfence by running the following commands: 

packagedir=libxshmfence-1.3
  docdir="--docdir=$XORG_PREFIX/share/doc/$packagedir"
  case $packagedir in
    libICE* )
      ./configure $XORG_CONFIG $docdir ICE_LIBS=-lpthread
    ;;

    libXfont2-[0-9]* )
      ./configure $XORG_CONFIG $docdir --disable-devel-docs
    ;;

    libXt-[0-9]* )
      ./configure $XORG_CONFIG $docdir \
                  --with-appdefaultdir=/etc/X11/app-defaults
    ;;

    * )
      ./configure $XORG_CONFIG $docdir
    ;;
  esac
  make
  #make check 2>&1 | tee ../$packagedir-make_check.log

Now as the root user: 

 make install
