Introduction to setxkbmap

Package Information

  • Download (HTTP): https://www.x.org/pub/individual/app/setxkbmap-1.3.2.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/app/setxkbmap-1.3.2.tar.bz2

  • Download MD5 sum: 93e736c98fb75856ee8227a0c49a128d

Installation of setxkbmap

Install setxkbmap by running the following commands: 

packagedir=setxkbmap-1.3.2
     case $packagedir in
       luit-[0-9]* )
         sed -i -e "/D_XOPEN/s/5/6/" configure
       ;;
     esac

     ./configure $XORG_CONFIG
     make

Now as the root user: 

 make install
