Introduction to xhost

Package Information

  • Download (HTTP): https://www.x.org/pub/individual/app/xhost-1.0.8.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/app/xhost-1.0.8.tar.bz2

  • Download MD5 sum: a48c72954ae6665e0616f6653636da8c

Installation of xhost

Install xhost by running the following commands: 

packagedir=xhost-1.0.8
     case $packagedir in
       luit-[0-9]* )
         sed -i -e "/D_XOPEN/s/5/6/" configure
       ;;
     esac

     ./configure $XORG_CONFIG
     make

Now as the root user: 

 make install
