Introduction to xsetroot

Package Information

  • Download (HTTP): https://www.x.org/pub/individual/app/xsetroot-1.1.2.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/app/xsetroot-1.1.2.tar.bz2

  • Download MD5 sum: 5fe769c8777a6e873ed1305e4ce2c353

Installation of xsetroot

Install xsetroot by running the following commands: 

packagedir=xsetroot-1.1.2
     case $packagedir in
       luit-[0-9]* )
         sed -i -e "/D_XOPEN/s/5/6/" configure
       ;;
     esac

     ./configure $XORG_CONFIG
     make

Now as the root user: 

 make install
