Introduction to xvinfo

Package Information

  • Download (HTTP): https://www.x.org/pub/individual/app/xvinfo-1.1.4.tar.bz2

  • Download (FTP): ftp://ftp.x.org/pub/individual/app/xvinfo-1.1.4.tar.bz2

  • Download MD5 sum: b13afec137b9b331814a9824ab03ec80

Installation of xvinfo

Install xvinfo by running the following commands: 

packagedir=xvinfo-1.1.4
     case $packagedir in
       luit-[0-9]* )
         sed -i -e "/D_XOPEN/s/5/6/" configure
       ;;
     esac

     ./configure $XORG_CONFIG
     make

Now as the root user: 

 make install
